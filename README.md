# ShortKit iOS SDK

Drop-in short-form video feeds for your iOS app.

## Installation

Add this package in Xcode via **File → Add Package Dependencies**:

```
https://github.com/shortkit/shortkit-ios
```

Or in your `Package.swift`:

```swift
.package(url: "https://github.com/shortkit/shortkit-ios.git", from: "1.0.0")
```

Full documentation at [docs.shortkit.dev](https://docs.shortkit.dev)
